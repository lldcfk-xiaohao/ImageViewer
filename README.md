## v2.0.0 — CFKViewer 风格重制

| 特性 | 说明 |
|------|------|
| ✨ 极简白色主题 | 告别工具栏，一切操作通过右键菜单完成 |
| 🖱 右键上下文菜单 | 复制图片/文件/路径、另存为、打印、打开位置、删除等 |
| ⚙️ 设置对话框 | 语言切换、幻灯片速度、启动选项 |

## 界面预览

### 欢迎页（CFKViewer 风格）
- 居中的图标 + 提示文字 + 打开按钮

- 底部状态栏带导航控件

## 功能特性

| 功能 | 说明 |
|------|------|
| 📂 打开文件 / 文件夹 | 单文件或整个文件夹批量加载 |
| ⬅ ➡ 导航 | 上一张 / 下一张，键盘左右方向键 |
| 🔍 缩放 | 鼠标滚轮 / 按钮放大缩小，最高 2000% |
| ⬜ 适应窗口 | 自动缩放至适合当前窗口大小 |
| 1:1 原始大小 | 恢复 100% 显示 |
| ↺ ↻ 旋转 | 左转 / 右转各 90° |
| 🖱 拖拽平移 | 鼠标左键拖拽移动图片 |
| ▶ 幻灯片播放 | 可自定义间隔时间自动播放 |
| ⛶ 全屏 | F11 切换全屏，ESC 退出 |
| 🗑 删除 | 一键删除当前图片（需二次确认） |
| ℹ 图片信息 | 分辨率、大小、格式、修改时间 |
| 📋 复制到剪贴板 | 图片 / 文件 / 路径 / 文件名 / 尺寸 |
| 💾 另存为 | 旋转后的图片可另存为新文件 |
| 🖨️ 打印 | 调用系统默认打印机打印 |
| 📁 在资源管理器中打开 | 快速定位文件位置 |
| 🔗 用其他程序打开 | 默认程序 / 画图 |
| 🔝 窗口置顶 | 始终在最前显示 |
| ⬚ 无边框模式 | 极致沉浸体验 |
| 🌐 多语言 | 简体中文 / English / 繁體中文 / 日本語 |
| EXIF 自动纠正 | 自动修正手机拍照的旋转方向 |
| 💾 配置持久化 | 语言、速度、启动选项自动保存 |

## 支持格式

`.jpg` `.jpeg` `.png` `.gif` `.bmp` `.webp` `.tiff` `.ico` `.ppm` `.pgm` `.pbm` `.pnm`

## 支持语言

| 代码 | 语言 |
|------|------|
| `zh_CN` | 简体中文（默认） |
| `en` | English |
| `zh_TW` | 繁體中文 |
| `ja` | 日本語 |

## 快速开始

### 从源码运行

```bash
pip install Pillow
python viewer.py
```

### 直接使用 EXE（推荐）

下载 [dist/CFKViewer.exe](dist/) 即可直接运行，无需安装任何依赖。

## 键盘快捷键

| 按键 | 功能 |
|------|------|
| `←` / `→` | 上一张 / 下一张 |
| `Space` | 下一张 |
| `↑` / `↓` | 放大 / 缩小 |
| `F11` | 全屏 |
| `ESC` | 退出全屏 / 退出无边框 / 退出程序 |
| `R` | 右转 90° |
| `L` | 左转 90° |
| `F` | 适应窗口 |
| `1` | 原始大小 (1:1) |
| `S` | 开始 / 停止幻灯片 |
| `Delete` | 删除当前图片 |
| `Ctrl+O` | 打开文件 |
| `Ctrl+C` | 复制图片到剪贴板 |
| `Ctrl+Shift+C` | 复制文件到剪贴板 |
| `Ctrl+S` | 另存为 |
| **鼠标右键** | 完整功能菜单 |

## 配置文件

首次运行后自动创建：

| 平台 | 路径 |
|------|------|
| Windows | `C:\Users\<用户名>\.cfkviewer\config.json` |
| Linux / macOS | `~/.cfkviewer/config.json` |

```json
{
  "language": "zh_CN",
  "slideshow_speed": 3,
  "reopen_last_folder": true
}
```
